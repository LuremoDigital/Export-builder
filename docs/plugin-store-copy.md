## Short Description

Saved export templates for Craft CMS: pick fields, filter precisely, schedule runs, and deliver by email or webhook. Accounting-ready Commerce order exports in Pro.

## Long Description

Export Builder turns exports from one-off requests into saved templates your whole team can run from the Control Panel.

**Why not just use Craft's native export?** Craft's element index export is fine for a quick one-time dump. Export Builder is for the exports you run more than once: month-end order reports for the bookkeeper, content audits, migration extracts, recurring data feeds. Save the field selection, column names, and filters once — then run it on demand, schedule it, or let it deliver itself.

### What you get

- Saved export templates for entries, users, categories, tags, and assets
- A field picker with native attributes, custom fields, relation paths, and practical Matrix paths — no code
- Rename and reorder columns; filter by section, site, status, keyword, dates, relations, and field values
- Small exports run instantly; larger ones queue safely, with downloadable run history
- Local export file retention cleanup (7 / 30 / 90 days)
- CSV and JSON output

### Pro

- **Commerce orders, products, and variants** — including an accounting preset: one row per completed order with revenue, recorded tax, discounts, shipping, refunds, and currency, ready for your accountant without spreadsheet cleanup
- Formie and Wheelform submission exports
- XLSX and Craft-native XML output
- Scheduled recurring exports
- Delivery by email, webhook, or Craft asset volume
- Template config import/export for moving saved templates between projects

### Built for

- Agencies fielding "can you send me an export of…" requests every month
- Commerce shops that need accountant-ready order data on a schedule
- In-house teams giving editors self-serve exports without developer time

### Positioning

Not a BI suite or integration platform — a focused export workflow that replaces repeated export scaffolding with a reusable Control Panel tool.
